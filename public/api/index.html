post /services

Forming new service entity.

  • Request
  • Response

Body

Media type: application/json

Type: object

Properties
  • testName: required(string)

    Name of service

  • testMethod: required(string)

    Standard method used

  • fieldOfTesting: required(string)

    branch of scientific category

  • sampleType: required(string)

    Matrix or media use in laboratory procedure

  • testFee: required(number)

    Amount of service rendered to client per service

Example:

{
    "testName": "Arsenic",
    "testMethod": "SM 200.7",
    "fieldOfTesting": "Environmental Toxicology",
    "sampleType": "Wastewater",
    "testFee": 94.44
}

HTTP status code 201

Service created.

HTTP status code 500

Error occurs in posting the entry.

Body

Media type: application/json

Type: any

Example:

{
    "code": "Internal",
    "message": "Service validation failed: testName: Path `testName` is required., testMethod: Path `testMethod` is required., fieldOfTesting: Path `fieldOfTesting` is required., sampleType: Path `sampleType` is required., testFee: Path `testFee` is required."
}
